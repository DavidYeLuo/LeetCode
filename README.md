## Note

## Build

Make

```sh
mkdir build
cd build
cmake ..
make -j4
```

Ninja
```sh
mkdir build
cd build
cmake .. -GNinja
ninja -j4
```

## Run All Tests

```sh
cd build/bin
./LeetCodeTests
```
